Position : Static

an element with position : static is not positioned in any special way
it is allways positioned according to the normal flow of the page.

this div element has position : static;

position : relative

an element with position : relative is positioned relative to brother element.

this div element has position : relative;

position: absolute

an element with position : absolute is positioned absolutely to its parent.

this div element has position : absolute;

position : fixed

an element with position : fixed is positioned relative to the viewport, while means it always staus in the same place, even if the page is scrolled, use top right bottom left to position.

this div element has position : fixed;
